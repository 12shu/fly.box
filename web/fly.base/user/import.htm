导入用户之前，请先 初始化部门 。

本系统只支持从Excel文件导入 ，请先下载用户导入模版文件，在模版文件中构建用户数据，为了避免
导入过程中出现异常，请不要调整模版文件结构，且严格遵循以下规则：

  • 不能修改列头或调整列的顺序。
  • 以“*”开头的列为必填信息，缺少将会导入失败。
  • 部门中填写的部门名称，必须是在系统中已创建。
  • 用户组中填写的用户组名称，必须是在系统中已创建，如不填写用户组，系统默认分配“普通用户”用户组。
  • 性别必须是“男”或者“女”。
  • 邮箱要符合邮件地址规则。
  • 生日必须是可识别的日期格式，如“1980-3-10”
