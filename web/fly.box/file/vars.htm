在Fly.Box自动创建文件夹时，可使用以下动态变量：

  • {姓名}：当前登录用户的姓名
  • {登录名}：当前登录用户的登录名
  • {部门名称}：当前登录用户所在部门的名称
  • {年}：当前年份
  • {月}：当前月份
  • {日}：当前日
  • {时}：当前时
  • {分}：当前分
  • {秒}：当前秒
  • {周}：星期几
如需要建立多级目录，可使用“\”或“/”分隔
如：
1.需要按自动创建文件夹“工作日报”，在工作日报目录下按日期自动创建目录，则可使用表达式“工作日报\{年}-{月}-{日}”
2.需要学生在交作业时按自己名称加日期创建文件夹，则可使用表达式“{姓名}\{年}-{月}-{日}”
